How to win rewards?
  1. Register yourself on the portal using this link and create your fantasy team.
  2. Predict daily match outcomes to earn cashback incentives.
  3. Participate in entertaining weekly events to receive surprises.
Points system
  1. A player receives 1 point for every run that is scored.
  2. A player receives 25 points for every wicket that is taken.
  3. Each point earned by your team's captain will be multiplied by 2.
  4. Each point earned by your team's vice-captain will be multiplied by 1.5.
  5. Each point earned by your player in a bonus match will be multiplied by 2.
  6. Correctly predictÂ a team to win the game to earn 20 points, while predict a tie to earn 75 points.
Daily prediction
  1. Maintain a streak of 5 correct prediction and receive straight cashback.
  2. If you incorrectly predict a game, you will restart at 0.
  3. If you miss a game, you will restart at 0.
  4. If a match is abandoned, streaks of users who made predictions will be kept; everyone else will have to start over at 0.
